अष्टाध्यायी से नाट्यशास्त्रम् — Classical Śāstric Culmination

The Classical Śāstras (600 BCE – 300 CE): Seven Disciplines, One Civilisational Project

Period Covered: 600 BCE – 300 CE (approx. 900 years)
Primary Sources: Aṣṭādhyāyī, Mīmāṃsāsūtra, Nyāyasūtra, Manusmṛti, Arthaśāstra, Carakasaṃhitā, Nāṭyaśāstra
Methodology: Internal textual analysis, comparative scholastic study, case-study analysis
Series Part: II of V
0.1Abstract

This paper examines the classical period of Sanskrit śāstric production — approximately 600 BCE to 300 CE — during which the Vedic auxiliary disciplines crystallised into seven fully autonomous systematic sciences: grammar (Vyākaraṇa), ritual hermeneutics (Mīmāṃsā), logic and ontology (Nyāya–Vaiśeṣika), jurisprudence (Dharmaśāstra), political economy (Arthaśāstra), medicine (Āyurveda), and aesthetics (Nāṭyaśāstra). The paper argues that these seven disciplines, despite their divergent subject matters, share a single methodological DNA traceable to the Prātiśākhya and Kalpa Sūtra traditions examined in Part I: the conviction that any domain, from the structure of words to the physiology of disease, can be fully described through a finite set of explicit, generative, hierarchically organised rules.

0.2Methodological Note

The analytical framework this paper applies across all disciplines is structural rather than content-specific: we ask, for each śāstra, the same four questions.

§ 1Pāṇini and the Aṣṭādhyāyī — Grammar as Axiomatic Science

The Most Ambitious Intellectual Project in the Ancient World: A Full Technical Anatomy

Pāṇini's Aṣṭādhyāyī ("Eight Chapters," c. 4th century BCE) is not a grammar book in any sense the modern world would recognise. It is a generative machine — a finite, closed set of 3,959 rules capable of producing, by sequential application, every grammatically well-formed Sanskrit expression from a primitive inventory of phonological and morphological atoms.

It is more precisely: the first documented instance of a complete, correct, formal specification of an infinite recursive set using a finite rule system — an achievement that the West did not independently arrive at until Chomsky's 1957 Syntactic Structures.

Th.1
Core Thesis

The Aṣṭādhyāyī as a Context-Sensitive Formal Language

After Frits Staal (1965), Paul Kiparsky (1982), George Cardona (1997), Rishi Rajpopat (2022)
"Pāṇini's grammar is a formal system in the mathematical sense: it has a primitive alphabet (the phonemes listed in the Śivasūtras), a set of explicit rules for rule-application, a meta-rule system governing rule precedence, and a domain of well-formed output."
Staal (1965): Demonstrated that the Aṣṭādhyāyī is a formal deductive system — its rules are not guidelines but axioms.
Kiparsky (1982): Showed that Pāṇini's rule-ordering conventions are equivalent to the feeding/bleeding typology independently developed by American phonologists in the 1960s.
Rajpopat (2022): Demonstrated that Pāṇini's own meta-rule 1.4.2 had been misapplied for 2,500 years. When correctly applied, every previously unresolved derivational conflict is resolved — proving the grammar was correctly and completely specified from the beginning.

The anubandha markers are metalinguistic tags invented by Pāṇini. Each tags the end of a group enabling pratyāhāra formation: ac = all vowels; hal = all consonants; yaṇ = semivowels. 41 pratyāhāras cover every natural class the grammar needs.

The First Metalanguage in Human History

The Śivasūtra system is the first documented instance of a deliberate distinction between an object language (Sanskrit) and a metalanguage (the grammar's own notational system) — a distinction that Frege did not formally articulate in the West until 1879.

§ 2Mīmāṃsā — Hermeneutics as Śāstra

The Science of Interpretation: When Reading a Text Became a Formal Discipline

Jaimini's Mīmāṃsāsūtra (c. 3rd–2nd century BCE) takes the act of reading — interpreting words, resolving ambiguity, determining what a text commands — and subjects it to exactly the same formal rigour that Pāṇini applied to grammar. It produces a complete theory of interpretation: a formal system from which, given any ambiguous or conflicting set of Vedic injunctions, a uniquely correct interpretation can be derived by rule-application.

The Foundational Problem

अथातो धर्मजिज्ञासा ।Jaimini, Mīmāṃsāsūtra 1.1.1 — "Now, therefore, the inquiry into dharma."

Type 1 — Apparent Redundancy

Two passages seem to say the same thing. Mīmāṃsā's answer: no two Vedic injunctions are genuinely redundant (apūrvatā principle) — if they appear so, one is a vidhi and the other is an arthavāda.

Type 2 — Direct Contradiction

Two passages prescribe mutually exclusive actions. Resolution hierarchy: specific beats general (viśeṣa beats sāmānya); later beats earlier (para beats pūrva); injunction beats commendation.

Type 3 — Incomplete Specification

A passage prescribes an action but leaves some element unspecified. Answer: by looking to the primary sacrifice (prakṛti) of which the current sacrifice is a modification (vikṛti).

Type 4 — Scope Indeterminacy

Mīmāṃsā's niyama, nitya, kāmya, and prāyaścitta classifications provide a complete taxonomy of obligation-types, resolving scope indeterminacy for every case.

2.2 The Five Classificatory Categories

1. Vidhi — The Injunction

The operative sentence-type: commands action. Subdivided into: utpattividhi (first introduction), niyamavidhi (restriction), parisaṃkhyāvidhi (explicit exclusion), adhikāravidhi (who is qualified). Only vidhis generate genuine dharmic obligation.

2. Arthavāda — The Commendation

Laudatory or explanatory text. Does not independently generate obligation. Three subtypes: stuty-arthavāda (praise), nindā-arthavāda (condemnation), parakṛti-arthavāda (narrative exemplar).

3. Mantra — The Operative Utterance

Sacred formulae whose meaning is subsidiary to their sonic function. They are performative: their utterance is itself the required action.

4. Nāmadheya — The Name/Label

Proper names of rituals, substances, or individuals. Purely referential — neither injunctive nor commendatory.

2.3 The Prābhākara and Bhāṭṭa Schools

Prābhākara School (after Prabhākara, c. 700 CE)

Core thesis: Obligation (kārya) is directly perceived in the verbal root itself. The verbal root denotes the obligatoriness of the action. Obligations are categorical (Kantian), not hypothetical.

Bhāṭṭa School (after Kumārila Bhaṭṭa, c. 700 CE)

Core thesis: Obligation is in a specific cognitive result: the recognition of a bhāvanā (causative force) that the sentence communicates. Obligation is instrumental-rational (Humean), not categorical.

CS·2A
Extended Case Study · Hermeneutics + Legal Philosophy

The Śyena Controversy — Full Analysis

Mīmāṃsāsūtra 1.3.4 — the complete debate

The Vedic text contains: śyenena abhicaran yajeta — "one wishing to harm an enemy should perform the Śyena (falcon) sacrifice." The Veda elsewhere enjoins general non-harm (ahiṃsā). The conflict is direct: one passage conditionally prescribes harm; another generally prohibits it.

Prābhākara Resolution

The Śyena injunction is a conditional vidhi (kāmyavidhi). The ahiṃsā injunction is an arthavāda. The law permits what morality condemns — this is the Prābhākara position, and it is internally consistent.

Bhāṭṭa (Kumārila) Resolution

The Śyena may be technically described in the Veda but is not dharmically sanctioned — performing it generates adharma. There is a meta-level constraint on all Vedic injunctions, and that constraint is the general principle of non-harm.

Śāstric Outcome

The Śyena controversy was never formally resolved within the Mīmāṃsā tradition — both positions are considered valid. This is not a failure of the śāstric method: it is its most honest output. When a genuinely contested normative question admits of two formally correct analyses from the same source, the śāstric tradition preserves both, marks them as distinct schools (pakṣas), and allows the question to remain open. This structural tolerance for principled disagreement is the Mīmāṃsā tradition's most underappreciated contribution.

§ 3Nyāya & Vaiśeṣika — Logic, Ontology, and the Architecture of Reality

India's Formal Logic and Atomic Theory: The Complete Epistemological and Ontological System

The Nyāya and Vaiśeṣika schools constitute ancient India's most systematic attempt to answer two foundational questions: How do we know anything? (Nyāya's epistemology and logic) and What is there? (Vaiśeṣika's ontology). Together they constitute a unified philosophical system whose scope and rigour rival anything produced by the Western philosophical tradition before the 17th century.

The Significance of Categories 10–16

Categories 10–16 have no parallel in the Western logical tradition before the 20th century. Nyāya's categories 10–16 constitute a complete theory of epistemic ethics in discourse: they specify what honest, truth-seeking intellectual exchange looks like (vāda), identify its degenerate forms (jalpa, vitaṇḍā), and enumerate every way a debater can betray the norms of inquiry.

3.2 The Four Pramāṇas

1. Pratyakṣa — Perception

"Cognition arising from the contact of a sense organ with its object, non-erroneous, and not dependent on words." Distinguishes indeterminate perception (nirvikapaka — bare sensory contact before conceptualisation) and determinate perception (savikalpaka — contact filtered through concepts). This anticipates Kant's distinction between intuition and concept.

2. Anumāna — Inference

The five-member syllogism plus the complete fallacy theory. Three types: pūrvavat (inferring from cause to effect), śeṣavat (inferring from effect to cause), sāmānyatodṛṣṭa (inferring by invariable concomitance).

3. Upamāna — Comparison

Knowledge arising from recognising a resemblance. Nyāya defends it: comparison produces genuinely new knowledge not available from perception or inference alone.

4. Śabda — Testimony

Knowledge arising from the utterance of a reliable speaker (āpta). Distinguishes dṛṣṭārtha śabda (testimony about observable matters) from adṛṣṭārtha śabda (testimony about non-observable matters — the Vedas, dharma). This is structurally equivalent to the contemporary debate between reductionist and anti-reductionist positions on testimony.

§ 4Dharmaśāstra — The Architecture of Legal Science

From Ritual Prescription to Jurisprudence: How India Built the Ancient World's Most Comprehensive Legal System

The Dharmaśāstra tradition represents a conceptual leap: from the Dharma Sūtras' school-specific prescriptions for conduct to universal, authoritative legal codes claiming jurisdiction over every domain of social life. No other ancient civilisation produced a body of legal literature comparable in scope, conceptual rigour, or longevity.

Extended Case Study · Jurisprudence

Dayabhāga vs. Mitākṣarā — The Complete Jurisprudential Analysis

Two schools, one question, two complete legal universes — still operative in Indian courts

The most consequential jurisprudential debate in the entire Dharmaśāstra tradition turns on a single question: At what moment does a son acquire a legally cognisable right in ancestral property?

Mitākṣarā — Birth-Right System

Basis: Vijñāneśvara reads Manusmṛti 9.104 as establishing co-ownership from the moment of birth. Consequences: A son can bring suit against his father during his lifetime to compel partition. A father cannot alienate ancestral property without the sons' consent. Property devolves by survivorship. Scope: All Indian states except Bengal and Assam.

Dāyabhāga — Death-Right System

Basis: Jīmūtavāhana distinguishes svāmitva (ownership) from svāmya (right to enjoyment on owner's death). The father is sole owner until death. Consequences: A father has absolute power of alienation during his lifetime. Property devolves by inheritance not survivorship. Scope: Bengal and Assam.

Living Legal Presence

The Hindu Succession Act 1956 (amended 2005) codified and reformed but did not abolish the Mitākṣarā coparcenary system. The fundamental structure remains operative and is litigated in Indian courts daily — two schools of 12th-century jurisprudential commentary on 2nd-century śāstric texts are the living basis of property law for approximately 80% of India's population.

§ 5Arthaśāstra — The Science of Statecraft

Kauṭilya's Arthaśāstra: Political Economy, Intelligence, Diplomacy, and War as Integrated Śāstra

Kauṭilya's Arthaśāstra (c. 4th–3rd century BCE in core composition) is a complete, integrated śāstra: a system that derives all political, economic, military, and diplomatic prescriptions from a set of explicitly stated first principles. No other ancient political text — not Thucydides, not Machiavelli, not even Hobbes — covers this range at this level of operational detail.

The Arthaśāstra's First Principles

सुखस्य मूलं धर्मः। धर्मस्य मूलं अर्थः। अर्थस्य मूलं राज्यम्।Arthaśāstra 1.7.3 — "The root of happiness is dharma. The root of dharma is artha. The root of artha is the kingdom." Statecraft is not the rival of dharma but its precondition.

The Seven Constituents (Saptāṅga)

Every state consists of exactly seven elements: (1) Svāmin — the king; (2) Amātya — ministers; (3) Janapada — territory and people; (4) Durga — fortified capital; (5) Kośa — treasury; (6) Daṇḍa — army; (7) Mitra — allies. Every political prescription in the Arthaśāstra can be located as an injunction about one of these seven elements.

The Six Foreign Policy Instruments (Ṣāḍguṇya)

Sandhi (peace treaty), Vigraha (war), Āsana (neutrality), Yāna (military mobilisation), Saṃśraya (seeking protection), Dvaidhibhāva (dual policy — simultaneous peace with one and war with another). Kauṭilya specifies exactly which maṇḍala configuration calls for which option.

Abhinavagupta's Sādhāraṇīkaraṇa — The Complete Account of Aesthetic Universalisation

How a sophisticated spectator can experience genuine aesthetic pleasure from witnessing suffering

The deepest question: how does a spectator who knows they are watching a performance experience genuine aesthetic pleasure from witnessing suffering? Abhinavagupta's answer introduces sādhāraṇīkaraṇa — "universalisation." The process operates in four phases:

Phase 1 — Decontextualisation
The Vibhāvas Strip the Emotion of Its Personal Particularity
The stage's vibhāvas act to remove the particularising conditions from the emotional content. What remains is grief-in-general — the essential structure of grief, freed from the accident of any specific griever's situation.
Phase 2 — Arousal of the Latent Vāsanā
The Generalised Emotion Activates the Spectator's Stored Emotional Impressions
The universalised emotional content resonates with latent emotional impressions (vāsanās), activating them without triggering the personal-particular associations that accompany them in ordinary life.
Phase 3 — The Aesthetic Distance
Theatrical Convention Creates a Protected Cognitive Space
The spectator's knowledge that the performance is a performance creates cognitive distance that prevents the universalised emotion from collapsing back into personal emotion. The spectator is moved but not overwhelmed.
Phase 4 — Rasa Itself
The Relishing of Universal Emotional Essence
What the spectator experiences is not grief but the universal form of grief — the essence of grief freed from its personal accidents, present to consciousness as an object of understanding rather than merely of feeling. Karuṇa-rasa is pleasurable precisely because it is grief universalised.

§ 11Chandas — The Science of Metre: Rhythm as Sacred Structure

Piṅgala's Chandaḥśāstra and the Mathematics Concealed Within Sacred Verse

Piṅgala's Chandaḥśāstra (c. 3rd–2nd century BCE) contains — embedded within its account of poetic metres — the earliest known description of a binary numeral system, a proto-Pascal's triangle, and a combinatorial method equivalent to what the West calls the Fibonacci sequence.

The Mathematical Discoveries Inside a Metrics Treatise

Binary Notation

Piṅgala assigns numerical values to sequences of L (light) and G (heavy) syllables: G = 0, L = 1. His algorithm for recovering a metre from its number is identical to the modern binary-to-decimal conversion algorithm. Binary arithmetic in the West: Leibniz (1703 CE). In India: Piṅgala (c. 3rd–2nd century BCE).

The Meru-Prastāra (Proto-Pascal's Triangle)

Piṅgala's method for computing the number of metres with exactly k heavy syllables produces a triangular arrangement where each entry is the sum of the two above it. Blaise Pascal published his Triangle in 1653 CE.

The Mātrāmeru (Fibonacci Sequence)

In computing the number of ways to scan a line of n morae, Piṅgala's algorithm generates the sequence 1, 2, 3, 5, 8, 13, 21… — the Fibonacci sequence. Leonardo Fibonacci published this sequence in 1202 CE.

Hemacandra's Independent Rediscovery

Hemacandra (1089–1172 CE) independently re-derives the same sequence in his Chandonuśāsana. In the West it is called the Fibonacci sequence; in India it is called the Mātrāmeru or Hemacandra sequence.

The nakṣatra system is older than the twelve-sign solar zodiac imported from Babylon (c. 5th century BCE). The sky is divided into exactly as many stations as the Moon takes days to complete one orbit — ritual calendar and astronomical precision are the same measurement problem.

18.1The Grammar as a Model of Biological Competence

The distinction Chomsky drew in 1965 between linguistic competence (the implicit knowledge of language structure every speaker possesses) and linguistic performance (the actual, error-prone use of language in real time) was revolutionary in Western linguistics. Pāṇini had made an equivalent distinction 2,300 years earlier — and had then done something Chomsky did not: he wrote the competence down, completely, in 3,959 rules.

The Aṣṭādhyāyī describes not what Sanskrit speakers say but what they know without knowing that they know it — the implicit, unconscious rule system that generates correct forms and rejects incorrect ones without deliberate reasoning. This is tacit knowledge made explicit. And making tacit knowledge explicit is, in cognitive science, one of the hardest problems there is.

How Pāṇini Accessed Tacit Knowledge — A Methodological Reconstruction

1
Grammaticality Judgment as Data
Native speaker intuitions about which forms are correct constitute the primary data set. Pāṇini systematically surveys the entire morphological space of Sanskrit and uses speaker judgments to distinguish grammatical from ungrammatical. This is exactly the methodology Chomsky later formalised as the empirical basis for generative grammar.
2
Minimal Pair Analysis
The Aṣṭādhyāyī's rules are defined by their environments — the immediately preceding and following phonological contexts. This requires identifying pairs of forms that differ in exactly one feature. The discovery that the rule applying in context A does not apply in the minimally different context B establishes the rule's domain boundary — the minimal pair methodology of modern phonology applied here to morphosyntax.
3
Exception as Falsifying Instance
Every apavāda (exception rule) in the Aṣṭādhyāyī is a falsifying instance for the general rule — evidence that the rule's scope must be restricted. Pāṇini treats exceptions not as irregularities to be memorised but as data points requiring explanatory accommodation. The grammar is progressively refined by exceptions, exactly as scientific theories are refined by anomalous data.
4
Derivational Testing
Each proposed rule must be tested by running the full derivation through the system and verifying that it produces the correct output and no incorrect outputs. A rule that over-generates or under-generates is wrong and must be revised. This is the generate-and-test methodology of computational linguistics — applied 2,300 years before the term existed.

The Śivasūtras as a Huffman Code — Optimal Compression of the Sanskrit Phoneme Inventory

Why Pāṇini's phoneme arrangement is the unique solution to a mathematical constraint problem

Huffman coding (David Huffman, 1952) constructs a binary code that minimises the expected description length of a message by assigning shorter codes to more frequent symbols. Pāṇini's pratyāhāra system achieves an analogous compression — not of individual symbols but of natural classes of phonemes — using a two-character encoding scheme that is demonstrably near-optimal.

The Compression Problem

Sanskrit has 52 distinct phonemes. A grammar's rules frequently need to refer to subsets of these phonemes. Without abbreviation, each reference requires listing all members of the class — making the grammar impossibly long to memorise. Pāṇini's solution: arrange all 52 phonemes in 14 groups such that every natural class used in the grammar rules is a contiguous subsequence of the arrangement. Name any such class with just two characters: first phoneme of the class + anubandha marker at the end.

The Optimality Result

Computational analysis by Rajpopat (2022) and Kiparsky confirms: the Śivasūtra arrangement is constrained to a very small set of valid permutations by the requirement that all grammatically relevant natural classes be addressable by pratyāhāra. This is the unique solution to a mathematical constraint problem with 52! possible arrangements — solved without combinatorial mathematics as a formal discipline, in 4th-century BCE Taxila.

Rajpopat 2022 — The Grammar is Self-Sufficient

When the correct meta-rule interpretation is applied, the grammar generates right forms without exception — Pāṇini had already solved every conflict that later commentators thought required additional ad hoc rules. This is the hallmark of a well-designed formal system: one that does not require external patches because it was correctly specified from the start. The Aṣṭādhyāyī has been in continuous use for 2,400 years and has never needed a bug fix.

Working Memory Architecture — Designed In

The sūtra style — maximum compression, minimum words — is not aesthetic preference. Pāṇini's rules were transmitted orally for centuries. Each sūtra had to be held in working memory while being applied in a derivation. The entire grammar is architected around the 7±2 chunk capacity of human short-term memory: the Śivasūtras encode phoneme classes as two-character abbreviations precisely because a student can hold two characters plus their referent as a single chunk. Shannon's source coding theorem (1948) establishes that optimal lossless compression encodes frequent patterns with short codes. Pāṇini's system does exactly this — 2,300 years before Shannon.

Arthavāda and the Descriptive–Normative Distinction in AI Training Data

How Mīmāṃsā's illocutionary taxonomy solves a class of value-learning errors in language models

One of the most persistent problems in training large language models is the conflation of descriptive statements ("X is the case") with normative statements ("X should be done") and evaluative statements ("X is good"). A model that cannot distinguish these will systematically generate normative content from descriptive premises — precisely the error Mīmāṃsā's arthavāda category was designed to prevent.

The Arthavāda Distinction in Linguistics

Mīmāṃsā distinguishes four types of passage by illocutionary force: vidhi (injunction — "do X"), mantra (operative speech), nāmadheya (name/label), and arthavāda (commendation — "X is good/praised"). Only vidhis generate binding obligations. Arthavādas cannot — they praise vidhis without independently enjoining action. This is a precise taxonomy of speech acts, anticipating Austin and Searle by 2,200 years.

The AI Training Problem This Solves

A model trained on text mixing vidhis and arthavādas without the distinction will learn to infer "you should do X" from "X is praised" — the arthavāda fallacy Mīmāṃsā explicitly blocks. A Mīmāṃsā-style illocutionary classifier as a preprocessing filter for training data would structurally prevent a class of value-learning errors that current RLHF approaches handle only approximately. The Five Hetvābhāsas as a Decision Tree for Invalid Inference

A computational reconstruction — five conditions, five failure modes, one complete validity test

Nyāya's five hetvābhāsas constitute a decision tree algorithm for detecting invalid arguments — a five-condition test that any proposed hetu must pass to be admissible as an inferential mark. The five conditions correspond precisely to the five components of a valid vyāpti relationship as systematised by Dharmakīrti. Each hetvābhāsa is the violation of exactly one condition — the taxonomy is complete (covers all possible failures) and exclusive (each failure belongs to exactly one category).

T1
Test for Savyabhicāra (Erratic Hetu)
Formal condition: Does H invariably co-occur with S — ∀x: H(x) → S(x)? Failure: ∃x: H(x) ∧ ¬S(x) — even one counterexample fails the test. Classic example: "Sound is impermanent because it is knowable" — but space is knowable and permanent. Modern equivalent: A predictor variable with less than 100% precision — savyabhicāra is the classical analogue of low precision in a classifier.
T2
Test for Viruddha (Contradictory Hetu)
Formal condition: Does H avoid invariably co-occurring with ¬S? Failure: ∀x: H(x) → ¬S(x) — the hetu proves the exact opposite of what was intended. Classic example: "Sound is permanent because it is produced" — but everything produced is impermanent. Modern equivalent: A feature with perfect negative predictive value assigned the wrong sign by the model.
T3
Test for Prakaraṇasama (Question-Begging Hetu)
Formal condition: Is H established independently of S? Failure: H ↔ S — accepting H already requires accepting S; the inference is circular. Classic example: "Sound is permanent because it is not non-permanent" — the hetu just restates the conclusion in negative form. Modern equivalent: Including the outcome variable as a predictor — the model predicts by using a disguised version of the answer (target leakage).
T4
Test for Sādhyasama (Unproved Hetu)
Formal condition: Is H more epistemically certain than S? Failure: P(H) ≈ P(S) — the hetu provides no evidential lift over the prior. Classic example: "Sound is impermanent because it is a quality" — but whether qualities are impermanent is precisely what is disputed. Modern equivalent: Zero information gain in feature selection; using a proxy variable as uncertain as the target variable itself.
T5
Test for Kālātīta (Mistimed Hetu)
Formal condition: Does H exist at the time the inference is made? Failure: H was observed at t₀ but the inference is drawn at t₁ > t₀ when H has ceased to obtain. Classic example: "The fire is hot because we saw smoke there yesterday" — the hetu is temporally disconnected from the current state being inferred. Modern equivalent: Concept drift — a model trained on t₀ distribution applied to a shifted t₁ distribution without retraining.
Algorithmic Completeness — A Type System for Inference

These five tests constitute a complete decision procedure: a hetu passing all five is valid; failing any one makes it a hetvābhāsa of that specific type. Nyāya's fallacy theory is, in modern terms, a type system for inference — each hetvābhāsa is a type error that prevents the inference from being well-typed. The classification is not merely a taxonomy of common mistakes; it is a formal proof that these five failure modes are exhaustive and mutually exclusive. No hetvābhāsa falls between categories or belongs to two categories simultaneously. This is the śāstric method applied to its own epistemological foundation.
